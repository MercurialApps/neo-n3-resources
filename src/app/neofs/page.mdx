# NeoFS

[NeoFS](https://fs.neo.org) is a distributed, decentralized object storage network developed by Neo SPCC.

- [NeoFS @ Neo Developers Portal](https://developers.neo.org/docs/n3/Advances/neofs/introduction/Concepts)
- [NeoFS REST Gateway](https://github.com/nspcc-dev/neofs-rest-gw)
- [NeoFS HTTP Gateway](https://github.com/nspcc-dev/neofs-http-gw)

## NeoFS Nodes

Public NeoFS nodes and current status can be found [here](https://status.fs.neo.org)
