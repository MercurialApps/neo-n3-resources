# Tooling

Various tools for developers on the Neo blockchain.

## Generic

- [neon-js](https://github.com/CityOfZion/neon-js)
  - Javascript libraries that allow the applications to interact with NEO blockchain
- [neo-wallet-adapter](https://github.com/rentfuse-labs/neo-wallet-adapter)
  - Modular TypeScript wallet adapters and components for NEO N3 applications.
- [Neonova](https://www.neonova.space)
  - Like Postman but for NEO N3.
- [neo-fairy](https://github.com/Hecate2/neo-fairy-test/)
  - Fairyland for Neo3 smart contract testing & debugging.
- [shrike](https://github.com/EdgeDLT/shrike)
  - Data analysis infrastructure for the Neo N3 blockchain.
- [neo-express & neo-trace](https://github.com/neo-project/neo-express)
  - Neo-Express is a privatenet optimized for development scenarios. Neo-Trace is a tool to generate Neo Smart Contract Debugger trace files for existing blocks or transactions.
- [neo3-privatenet-docker](https://github.com/AxLabs/neo3-privatenet-docker)
  - Docker images for running a Neo N3 private network, in record time!
- [Neo N3 Vanity Address Generator](https://github.com/yildizoglu/NeoN3AddressGenerator)
- [Neo Blockchain Toolkit](https://marketplace.visualstudio.com/items?itemName=ngd-seattle.neo-blockchain-toolkit)
  - Develop, debug, test, deploy, track and manage your blockchain applications
- [N3 Data Converter](https://developers.neo.org/converter)
  - Perform conversions between the various data types supported on the Neo blockchain
- [N3 Testnet (T5) Faucet](https://n3t5wish.ngd.network/#/)

## C#

- [neo-test](https://github.com/ngdenterprise/neo-test)
  - Neo N3 smart contract unit test framework.
