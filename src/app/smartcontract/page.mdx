## Token Standards

There are 2 main token standards:

[Non-Fungible Token Standard (NEP-11)](https://github.com/neo-project/proposals/blob/master/nep-11.mediawiki)

[Token (Fungible) Standard (NEP-17)](https://github.com/neo-project/proposals/blob/master/nep-17.mediawiki)

**NEP-11 Examples:**

- Java: [neow3j - Implementing a NEP-11 (NFT) Smart Contract in Java](https://developers.neo.org/tutorials/2023/08/10/neow3j-nep11)
- NEP-11 Smart Contract Templates
  - [Java](https://github.com/neow3j/neow3j-examples-java/blob/master/src/main/java/io/neow3j/examples/contractdevelopment/contracts/NonFungibleToken.java) from AxLabs
  - [Python](https://github.com/CityOfZion/neo3-boa/blob/master/boa3_test/examples/nep11_non_divisible.py) from CoZ
  - [Go](https://github.com/nspcc-dev/neo-go/blob/master/examples/token/token.go) from NSPCC
  - [C#](https://github.com/neo-project/neo-devpack-dotnet/blob/master/src/Neo.SmartContract.Framework/Nep11Token.cs) from Neo
  - [TypeScript](https://github.com/ngdenterprise/neo-devpack-ts/blob/main/samples/nep11token/nep11token.ts) from NGD
